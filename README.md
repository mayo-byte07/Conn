# 🚀 Conn | Premium Link-in-Bio SaaS

Welcome to **Conn**, a next-generation link-in-bio platform built to elevate digital branding for creators, professionals, and businesses. Move beyond the standard alternatives with a cutting-edge visual experience and a robust admin dashboard! ✨

---

## ✨ Key Features

- 🎨 **Breathtaking UI/UX:** Sophisticated 3D tilt-responsive cards, dynamic aurora background animations, glassmorphism, and custom magnetic cursor effects.
- 🔐 **Secure Authentication:** Full user login/registration system keeping your data and dashboard completely secure.
- 🎛️ **Powerful Admin Dashboard:** Intuitive control over your digital identity. Effortlessly add, reorder, and manage unlimited links and profile details.
- 💳 **Tiered Subscriptions:** Built-in monetization (Free, Plus, and Pro tiers) integrated seamlessly with the **Razorpay** payment gateway to gate premium features.
- 📈 **Built-in SEO & Analytics:** Optimized for search engines out-of-the-box, acting as an interactive, high-converting digital business card.
- 📱 **Mobile-First Layout:** Engineered to look pristine and function flawlessly on any screen size.
- 🖌️ **Deep Customization:** Unlock exclusive visual themes, custom CSS, and advanced styling options as a premium user.

## ⚙️ Quick Start

Follow these simple steps to get a local copy up and running.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/conn.git
   cd conn
   ```

2. **Install project dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file in the root directory and add your necessary API keys and secrets:
   ```env
   PORT=3000
   SESSION_SECRET=your_super_secret_key
   RAZORPAY_KEY_ID=your_razorpay_key_id
   RAZORPAY_KEY_SECRET=your_razorpay_key_secret
   ```

4. **Run the application**
   ```bash
   npm run dev 
   # or npm start
   ```

5. **Open your browser**
   Navigate to `http://localhost:3000` 🌐
